1. **Install VS Code Extensions:**
   - Open VS Code
   - Install "CMake Tools" by Microsoft
   - Install "C/C++" by Microsoft

2. **Open the project:**
   ```bash
   cd /path/to/Quadtree
   code .
   ```

3. **Configure and build in VS Code:**
   - Press `Ctrl+Shift+P` (or `Cmd+Shift+P` on macOS)
   - Type "CMake: Configure" and press Enter
   - Select your preferred compiler when prompted
   - Wait for configuration to complete

4. **Build the project:**
   - Press `Ctrl+Shift+P` (or `Cmd+Shift+P` on macOS)
   - Type "CMake: Build" and press Enter
   - Or click the build button in the bottom status bar

5. **Run the application:**
   - Press `Ctrl+F5` (or `Cmd+F5` on macOS) for debugging
   - Or run from the bottom status bar
